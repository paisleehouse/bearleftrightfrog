snack attack

Published on January, 7 2025

a few months ago i got the idea in my head that i wanted to try and make a video game, so i'm trying it out! i used the MagiPi Code the Classics book as a starting point and used their Cavern Master python code to create a game called Snack Attack. i've loved the process so much that i want to take the game to the next level, which means i'm in the process of learning Godot.


in addition to designing the game, i'm making the characters and all of the sounds/music. hopefully something playable happens in the next year!
